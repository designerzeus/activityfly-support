Does ActivityFly work without Pro?

Yes. CPU, RAM, disk, and network monitoring are available in the free version.

Is Pro a subscription?

No. ActivityFly Pro is a one-time lifetime purchase.

Why is Battery hidden on some Macs?

Battery features are shown only on Macs with a battery. Desktop Macs such as Mac mini do not show the battery module.

Privacy Policy

No user tracking

ActivityFly does not track users, does not show ads, and does not require an account.

Local system information

ActivityFly displays system information such as CPU usage, memory usage, disk capacity, network speed, battery details, temperatures, and active processes locally on your Mac.

Data collection

The developer does not collect, store, or sell personal data from ActivityFly. System metrics are used to display information inside the app.

Network information

ActivityFly may show local network details and public IP information when available. This information is displayed to the user and is not stored by the developer.
